## 使用方法
使用前需要在config中填入有效CIPHERTEXT字段内容

首次使用需要手动登录一次保存有效COOKIES

## 功能简介
获取近N天内，某城市某关键词的百度指数结果。（相关参数在config等文件中进行配置）

保存文件格式有两种，.xlsx格式或者.json格式。

指数结果是全量数据，没有区分移动端和PC端，需要相关数据可以自行修改相关参数。

目前一次搜索结果支持的最大关键词数量为5个。
